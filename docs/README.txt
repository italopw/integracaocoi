Como rodar o site corretamente com Go Live (VS Code)
====================================================

1. Feche o VS Code completamente.
2. Abra o VS Code novamente.
3. No menu, clique em “Arquivo” > “Abrir Pasta...” (ou “File” > “Open Folder...”).
4. Navegue até:
   C:\Users\Ian Italo\OneDrive\integracaoCoi\docs
5. Selecione a pasta docs e clique em “Selecionar Pasta”.
6. Agora, com a pasta docs aberta como raiz do projeto no VS Code:
   - Clique no botão “Go Live” no canto inferior direito.
   - O Live Server vai servir apenas os arquivos dentro de docs (sem duplicação ou caminhos errados).

Se seguir esses passos, o Go Live funcionará corretamente e seu site será exibido normalmente.

Se precisar de ajuda, peça para o Copilot novamente!
